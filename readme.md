# ESG AI Dashboard 🌍

This project is an AI-powered ESG (Environmental, Social, and Governance) dashboard built using React.js.  
The dashboard helps companies monitor emissions, analyze ESG performance, and visualize sustainability data through interactive charts and AI-driven insights.

I built this project to explore modern dashboard UI design, React development, data visualization, and sustainability analytics concepts.

---

## Features

- Dark / Light Mode
- ESG Performance Score
- Interactive Bar Chart
- Pie Chart Analytics
- AI-Based Sustainability Recommendations
- Risk Detection System
- Real-Time Company Search
- Responsive Dashboard Design
- Modern Glassmorphism UI
- Live ESG Alerts

## Dashboard Sections

### ESG Performance Score
Displays an overall ESG score using a circular progress design.

### Emission Analytics
Shows company emission data using interactive bar charts.

### ESG Distribution
Visualizes sustainability distribution using pie charts.

### AI Recommendations
Provides suggestions to improve ESG performance.

### Risk Detection
Detects emission risk levels like Low, Medium, and High Risk.

## Installation

Clone the repository:

```bash
git clone https://github.com/joshiviveksharma/ESG-AI-Dashboard.git
```

Move into the project folder:

```bash
cd ESG-AI-Dashboard
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

The project is deployed using Vercel.
